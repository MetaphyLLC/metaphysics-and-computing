## 📌 About

This repository hosts all custom-built HTML pages for the Metaphysics and Computing website. Each page is a self-contained HTML file featuring:

- Dark theme with teal (`#3D8C8C`) + gold (`#C9A84C`) color palette
- Animated particle canvas background
- Scroll-progress nav bar
- Hero section with sacred geometry SVG overlay
- Accordion / reveal card layout for content
- Grok AI integration (expandable belief/concept panels)
- Responsive design · Google Fonts (Playfair Display, Inter, JetBrains Mono)

## 🔧 Page Template Standard

All custom pages follow the pattern established by the three reference pages (Tachymetric Manifesto, Architecture of Harmony, Cryptography of Being):

1. **Self-contained** — one `.html` file, no external CSS/JS dependencies beyond Google Fonts CDN
2. **Particle animation** — floating dot canvas via embedded JS
3. **Reading progress bar** — top-of-nav gradient bar
4. **Hero section** — fullscreen, video background (Morbius Lemniscate QEGG), sacred geometry SVG overlay
5. **Accordion cards** — collapsible content sections with Grok AI explore links
6. **Scroll reveal** — `IntersectionObserver`-driven entrance animations
